# 예제 - 호텔 예약

## 서비스 시나리오


- 기능적 요구사항
1. 고객이 객실을 선택하여 예약 요청한다.
2. 고객이 결제한다.
3. 결제가 완료되면 호텔에 예약 내용이 전송된다.
4. 객실이 있다면 예약처리가 완료되며 객실 상태를 업데이트한다.
5. 남은 객실이 없다면 결제 환불 단계로 비즈니스 롤백하여 환불 완료 후 객실 상태를 업데이트한다.
6. 고객이 마이페이지를 조회한다.
7. 고객이 예약내역을 취소할 수 있다.
